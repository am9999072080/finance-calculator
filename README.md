* Веб-приложение на Java с интеграцией Swagger UI с возможностью вводить данные непосредственно в браузере через HTML страницу которое позволяет вводить финансовые показатели и получать расчет чистого баланса или чистой прибыли.
<img width="256" height="256" alt="image" src="https://github.com/user-attachments/assets/4587d6e6-51ba-4773-b173-4ea80bd52693" />
* Пользователь открывает браузер и вводит адрес сервера (например, http://localhost:8080).
* Появляется страница с формой, куда можно внести значения активов, обязательств, дохода и расхода.
* После нажатия на кнопку "Рассчитать чистый баланс", данные отправляются на сервер.
* Сервер обрабатывает запрос и возвращает рассчитанный результат обратно в браузер.
* Результат выводится прямо на той же странице в соответствующем поле.
